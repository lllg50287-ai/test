Структура проекта:

ansible/
├── playbook.yml
├── roles/
│   └── todo-api/
│       ├── tasks/main.yml
│       ├── templates/
│       │   └── todo-api.service.j2
│       └── vars/main.yml

Описание каждого файла:
   playbook.yml:      
       Описание хоста, роли (todoapp)
    roles/todoapp: 
        tasks: 
            main: Переменные (юзер, группа, директория, название файла, порт, хост, путь к базе,                среда, название сервиса). Скрипт на создание (группа, системный юзер, директория,                  копия файла, файловый сервис (reload), запуск сервиса)
        templates:
            файл сервиса
        vars:
            порт, имя, название файла
